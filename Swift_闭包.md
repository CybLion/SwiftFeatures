#Swift闭包（Closure）

> 闭包是一种可以在代码中作为参数传递，自含的功能块。 闭包类似于 C 和 Objective-C 中的 block 语法 又或者类似于其他语言中的 lambda 语法。闭包是一种灵活的代码块，能作为参数传递，提高代码的灵活性，将一些自定义功能，交给闭包来完成。

##语法

一般闭包的形式如下：
<pre lang=swift>
	{ (parameters) -> returnType in
	    statements
	}
</pre>
